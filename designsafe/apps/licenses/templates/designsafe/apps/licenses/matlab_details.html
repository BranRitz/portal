Mathworks' MATLAB is available for use in the DesignSafe Discovery Workspace but requires a license before you can use the application. MATLAB is supported under a "bring your own license" model.

In order to run MATLAB at DesignSafe using "bring your own license", you need:

  • a MATLAB floating network license; and
  • a MATLAB license server that allows remote access from computational resources used by DesignSafe.

If you are unsure about the above details, please CC your MATLAB license administrator on the request.
