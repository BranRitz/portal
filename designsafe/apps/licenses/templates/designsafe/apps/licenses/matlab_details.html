Mathworks' MATLAB is available for use in the DesignSafe Discovery Workspace but requires a license before you can use the application. MATLAB is supported under a "bring your own license" model. In order to run MATLAB at DesignSafe, your license must satisfy the following requirements:

  • Be supported for FLEXlm license management.
  • The FLEXlm license server must be accessible from the computational resources used by DesignSafe in order to validate the license.
